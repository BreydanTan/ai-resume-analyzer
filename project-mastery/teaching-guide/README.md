### 五个学习阶段（按顺序学）

## 💡 使用建议

### 最佳学习方式

1. **先读，再做**
   - 先完整读一遍（不用全懂）
   - 然后按步骤操作
   - 遇到不懂就往回查

2. **多用AI协助**
   - 复制提示词，粘贴给Claude/ChatGPT
   - 让AI讲解代码
   - 但要自己理解，不是直接复制答案

3. **做完一个阶段要复习**
   - 闭上眼睛，能否说出关键概念？
   - 用自己的话记录学到了什么
   - 教别人来检验理解

### 学习节奏

**快速模式**（有编程基础的人）:
```
第1-2周: 第1-3阶段 (基础)
第3周: 第4阶段 (核心)
第4周: 第5阶段 (扩展)
```

**标准模式**（推荐）:
```
第1周: 第1-2阶段
第2周: 第3阶段
第3-4周: 第4阶段
第5-6周: 第5阶段 + 完成挑战
```

**深度模式**（想完全掌握）:
```
每个阶段停留2-3周
多做挑战和练习
深入理解每个概念
```
