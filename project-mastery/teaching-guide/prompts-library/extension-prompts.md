# 功能扩展提示词 | Extension Prompts

## 添加新页面

### 提示词1: 创建新页面

```
我想在AI简历分析项目中添加一个新页面: [页面名]

功能描述:
[详细描述这个页面做什么]

路由:
- 路径: [URL]
- 显示条件: [何时显示这个页面]

界面需求:
[描述页面应该显示什么]

数据需求:
[需要什么数据，从哪里获取]

能给我:
1. 新的 .tsx 文件的模板代码
2. 需要修改的现有文件列表
3. 新增的组件需求
4. 可能的问题和解决方案
```

### 提示词2: 添加新菜单项

```
我想在导航菜单中添加一个新的菜单项: [菜单名]

要求:
- 菜单文本: [文本]
- 指向页面: [URL]
- 图标: [如果需要]
- 显示条件: [何时显示]

能给我:
1. 在 Navbar 中添加的代码
2. 路由配置的改动
3. 样式设置（如果需要）
4. 测试步骤
```

## 通用扩展模板

```
我想给项目添加: [功能名]

需求分析:
1. 做什么: [功能描述]
2. 怎么做: [用户操作流程]
3. 显示在哪: [UI位置]
4. 数据怎么来: [数据来源]
5. 结果怎么用: [输出处理]

技术细节:
1. 需要的库: [如果有]
2. 影响的文件: [会改哪些文件]
3. 可能的问题: [预见的困难]

能给我:
1. 完整的实现方案
2. 代码示例
3. 集成指南
4. 测试方法
5. 可能的优化方向
```
