## 🚀 实践任务

### 任务1️⃣：安装开发工具（15-20分钟）

这一步是为你的电脑"装工具"。就像做饭前要装好刀和砧板。

#### 步骤1: 检查是否有Node.js

打开"终端"或"命令提示符"：

**Windows用户**:
- 按 `Windows键 + R`
- 输入 `cmd` 然后按Enter
- 会弹出黑色窗口

**Mac用户**:
- 按 `Command + Space`
- 输入 `terminal` 然后按Enter

**Linux用户**:
- Ctrl + Alt + T 打开终端

#### 步骤2: 输入检查命令

在终端中输入：
```bash
node --version
```

然后按Enter。

**会看到什么?**

**情况A** (已安装):
```
v20.10.0
```
太好了！你已经有了Node.js，跳到下一步。

**情况B** (未安装):
```
'node' is not recognized...
或
command not found: node
```
继续下面的安装步骤。

#### 步骤3: 安装Node.js（如果没有）

访问: https://nodejs.org

1. 点击左边的 **LTS** (推荐版本)
2. 下载对应你操作系统的安装程序
3. 双击安装，一路点"Next"即可
4. 安装完成后，**重启电脑**
5. 再次在终端输入 `node --version` 确认安装成功

**预计用时**: 5-10分钟

---

### 任务2️⃣：克隆项目（5-10分钟）

这一步是"下载代码"。

#### 步骤1: 打开终端，选择工作目录

选一个你喜欢放项目的地方（比如桌面或"我的文档"）。

```bash
# 进入桌面 (Windows)
cd Desktop

# 或进入Documents
cd Documents

# 或回到家目录
cd ~
```

#### 步骤2: 克隆项目

```bash
git clone https://github.com/your-repo/ai-resume-analyzer.git
cd ai-resume-analyzer
```

输入命令后等待下载完成（可能需要1-2分钟）。

**成功标志**:
- 看到 `Cloning into 'ai-resume-analyzer'...`
- 看到 `done.`
- 回到命令提示符

#### 步骤3: 验证下载成功

```bash
ls
# 或在Windows中:
dir
```

你应该看到：
```
README.md
package.json
app/
public/
...
```

**预计用时**: 2-3分钟

---

### 任务3️⃣：安装依赖（10-15分钟）

这一步是"下载项目需要的所有零件"。

#### 步骤1: 确保你在项目目录

```bash
pwd
# 应该显示: .../ai-resume-analyzer

# 如果不在，进入目录
cd ai-resume-analyzer
```

#### 步骤2: 安装依赖

```bash
npm install
```

看到大量信息打印，这是正常的。

**这个过程会**:
- 下载 package.json 中列出的所有库
- 创建 node_modules 文件夹 (会很大，几百MB)
- 完成后回到命令提示符

**预计用时**: 5-10分钟 (取决于网速)

**完成标志**:
```
added 500+ packages, and audited 501 packages in 2m
```

---

### 任务4️⃣：启动开发服务器（5分钟）

这一步是"打开应用"。

```bash
npm run dev
```

输入这个命令。看到输出：

```
  VITE v6.3.3  dev server running at:

  ➜  Local:   http://localhost:5173/
  ➜  press h + enter to show help
```

这说明成功了！

---

### 任务5️⃣：在浏览器打开（1分钟）

现在打开浏览器（任何浏览器都可以）：

1. 在地址栏输入：`http://localhost:5173`
2. 按Enter

**会看到什么?**

一个漂亮的网页，可能显示：
- "AI Resume Analyzer" 标题
- 登录按钮
- 一些其他内容

🎉 **恭喜！项目运行起来了！**

---

### 任务6️⃣：修改一个文字（5分钟）

现在我们修改点东西，看看"自动刷新"的魔力。

#### 步骤1: 打开你的代码编辑器

推荐用 **VS Code** (免费，很好用)：
- 下载: https://code.visualstudio.com
- 安装好后，打开编辑器

#### 步骤2: 打开项目文件夹

在VS Code中：
1. File → Open Folder
2. 选择 `ai-resume-analyzer` 文件夹
3. 点打开

现在你应该能看到文件夹结构。

#### 步骤3: 找到一个简单的文件

在左边的文件树中：
```
ai-resume-analyzer/
├── app/
│   ├── routes/
│   │   └── home.tsx  ← 打开这个
```

点击 `home.tsx` 打开它。

#### 步骤4: 找到可以改的文字

在 `home.tsx` 中，找到这样的代码：

```typescript
<h1>Resume List</h1>
或
<p>Welcome to...</p>
或任何英文文字
```

#### 步骤5: 改一个字

比如把：
```
<h1>Resume List</h1>
```

改成：
```
<h1>我的简历列表</h1>
```

保存文件（Ctrl+S 或 Cmd+S）。

#### 步骤6: 看神奇的事发生

回到浏览器。**不用按F5刷新！**

浏览器会自动刷新，你会看到改后的文字！

```
之前: Resume List
现在: 我的简历列表
```

🎊 **这就是"热更新"(HMR)的魔力！**

## 常见问题 FAQ

### Q1: 为什么需要Node.js？
**A**: Node.js提供了 npm (包管理器) 和开发服务器。不用它，我们就没法运行项目。

### Q2: 如果我用Windows，命令会不同吗？
**A**: 大部分命令一样。但如果看到"命令不识别"，试试：
```bash
# 改这个
npm run dev

# 试试
npm start
```

### Q3: 可以关闭终端吗？
**A**: 不行！终端窗口要一直开着。终端运行的服务器让浏览器能访问。如果关了，网页就不能工作了。

### Q4: 如果改错了怎么办？
**A**:
- 改回原来的内容 → 保存 → 浏览器自动刷新就好了
- 或者关闭文件不保存 (Ctrl+K, Ctrl+U)

### Q5: 要学会 Git 和 GitHub 吗？
**A**: 现在不需要。你已经克隆了项目，够用了。后面如果要上传改动，再学 Git。

### Q6: "npm" 是什么？
**A**: Node Package Manager，用来下载和管理代码库的工具。就像手机的"应用商店"，但给程序员用。

### Q7: 可以用其他编辑器吗？
**A**: 可以！VS Code 最好用，但也可以用 Sublime Text, Atom, WebStorm 等。

### Q8: 打开浏览器时，出现"无法连接"？
**A**: 检查：
1. 终端还在运行吗？(如果自己关了，`npm run dev` 再启动)
2. 地址栏输对了吗？(http://localhost:5173，注意冒号和数字)
3. 防火墙阻止了吗？(允许本地连接)
