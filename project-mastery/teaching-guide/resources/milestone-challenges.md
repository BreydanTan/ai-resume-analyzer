## 挑战1️⃣：修改欢迎信息 (初级)

### 难度: ⭐ 容易
### 估计时间: 15分钟
### 学习要点: 阅读代码、修改React组件

### 🎯 任务
在首页(/)显示一条个性化的欢迎信息。

### 📝 需求

1. 找到首页的代码文件
2. 找到显示"欢迎"之类文本的地方
3. 改成自己的欢迎信息，比如：
   ```
   "欢迎来到AI简历分析系统！👋"
   ```

4. 在登录后的首页显示用户名：
   ```
   "欢迎，[用户名]！"
   ```

### 📊 验证清单

- [ ] 能找到首页代码 (routes/home.tsx)
- [ ] 能找到欢迎信息显示的位置
- [ ] 能修改文本
- [ ] 能访问当前用户信息
- [ ] 显示的内容刷新后还在
- [ ] 没有破坏其他功能

### 💡 提示

```typescript
// 获取当前用户信息
const { auth } = usePuterStore()
const username = auth.user?.username

// 在JSX中显示
<h1>欢迎, {username}!</h1>
```

### ✅ 成功标志

- 首页显示自定义欢迎信息
- 显示了用户名
- 浏览器刷新后内容不变

### 🤖 AI求助提示词


## 挑战3️⃣：添加简历统计 (中级)

### 难度: ⭐⭐⭐ 中等
### 估计时间: 1小时
### 学习要点: 数据处理、计算

### 🎯 任务
在首页显示简历统计信息。

### 📝 需求

显示：
1. 已上传的简历总数
2. 平均评分
3. 最高分和最低分
4. 不同评分段的数量 (90+, 80-90, 70-80, <70)

### 示例输出

```
📊 统计信息
总简历数: 5份
平均分: 82.4
最高分: 92
最低分: 71

评分分布:
🟢 90+分: 1份
🟡 80-90分: 2份
🟠 70-80分: 1份
🔴 <70分: 1份
```

### 📊 验证清单

- [ ] 能获取所有简历数据
- [ ] 能正确计算总数
- [ ] 能正确计算平均分
- [ ] 能正确找最高和最低
- [ ] 能正确分类统计
- [ ] 显示在适当位置

### 💡 提示

```typescript
const calculateStats = (resumes: Resume[]) => {
  const total = resumes.length
  const avg = resumes.reduce((sum, r) =>
    sum + r.feedback.overallScore, 0) / total

  const max = Math.max(...resumes.map(r =>
    r.feedback.overallScore))
  const min = Math.min(...resumes.map(r =>
    r.feedback.overallScore))

  return { total, avg, max, min }
}
```

### ✅ 成功标志

- 统计信息准确
- 显示在合适的位置
- 数据实时更新

4. 实现上述4个统计内容
